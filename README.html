    Introduction

    Our game will be a shooting range game. There will be a target on a single colored background where the user will click the target to get points. Just like darts, where you click matters. Part darts, part wack-o-mole.

    Functional Requirements

  • Menu (start, settings, quit)
  • Settings Menu(Difficulty, Background Color)
  • Multiple difficulties (Easy, Medium, Hard)
  • Different backgrounds
  • High score tracker
  • Game modes (regular, speed run(not completed yet))
  • Timer for time limit to click target
  • Points based on target acccuracy

    Non-Functional Requirements

  • Not too cpu-heavy
  • Programmed in Java
    • Kept up to date with future versions

    System Evolution

  • Will support upcoming Java versions.
